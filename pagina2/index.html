Para entender mejor este concepto, es importante que establezcamos las diferencias entre una emisión en directo y un streaming . Cuando hablamos de una emisión en directo, hablamos del envío de una señal desde un punto. Los receptores captan las ondas, las descodifican y muestran el contenido. Como te podrás imaginar, este es el sistema que se emplea para emitir desde la radio o la televisión. Para ver o escuchar un contenido en concreto, el usuario idebe encender el aparato a la hora y la fecha en la que se emite.
